# Tesla 供应链简洁报告

**截至 2026-05-27，基于公开资料**

## 1. 整体判断

Tesla 的供应链整体看起来**仍具备较强规模化和全球本地化能力**，但风险集中在几个上游环节：**电池、电池材料、中国中游供应链、关税/出口管制，以及 AI 算力芯片**。Tesla 自己也在年报中承认，其产品来自全球数千家供应商，部分零部件仍存在单一来源风险；电池方面则只完全认证了很有限数量的供应商，切换弹性有限。

## 2. 最关键的供应链环节和公司

## 3. 证据比较强的判断

**电池是 Tesla 最核心的外部供应依赖。** 这是 Tesla 自己在年报中明确披露的风险：其依赖锂离子电芯，且完全认证的供应商数量很少，供应商无法按量、按质、按价交付时，可能影响产量或成本。

**中国供应链对 Tesla 很重要。** 上海工厂的本地化比例和中国供应商规模，说明中国不只是 Tesla 的销售市场，也是关键制造和零部件基地。([Reuters][3])

**上游材料和政策风险是真实存在的。** Tesla 年报提到关税和贸易反制影响供应链成本；IEA 也指出，中国在电池材料、石墨、LFP 正极和部分关键矿物加工环节占比很高，出口管制可能影响汽车、电池和 AI 相关产业。

## 4. 目前只能算线索、不能下结论的部分

**不能精确断定每个车型、每个工厂的供应商份额。** 公开资料能确认 Panasonic、CATL、LGES 等重要电池供应商，但无法可靠拆出“某车型某工厂某供应商占比”。

**4680 电池和高镍材料链仍有不确定性。** L&F 与 Tesla 相关的高镍正极材料合同金额大幅下调，这是明确信号；但外界关于原因的解释，例如 4680 量产难度、Cybertruck 需求、干电极工艺问题等，部分来自分析师和消息源，不能全部当作公司确认事实。([Reuters][4])

**Samsung、Nvidia 是重要跟踪对象，但不要过度推断。** 公开报道显示 Tesla 与 AI 芯片、训练算力有关的依赖在上升；但具体采购量、交付节奏、良率和合同条款没有充分公开。([Reuters][2])

**BYD 等潜在供应商不宜直接列为已确认核心供应商。** 有过媒体和行业讨论，但在本轮可核验的核心公开材料中，不足以把它作为 Tesla 当前关键供应商写入确定结论。

## 5. 最值得继续追踪的上游环节

最值得盯的是 **LFP 电池链、石墨负极、正极材料、锂精炼、镍材料、AI 芯片制造与先进封装**。其中 LFP 对低成本车和储能都很关键；石墨、正极材料和 LFP 中游高度集中在中国；锂价和锂精炼能力会影响电池成本；AI 芯片则关系到 FSD、Robotaxi 和 Optimus 的推进速度。([IEA][1])

## 6. 下一步研究建议

1. **按产品线拆供应链**：把 Model 3/Y、Cybertruck、Semi、Megapack、Powerwall 分开研究，不要混成一个 Tesla 总供应链。
2. **按工厂拆供应链**：上海、柏林、得州、内华达、加州、储能工厂的供应商结构可能不同。
3. **重点核验电池供应**：持续看 Panasonic、CATL、LGES、Tesla 自有 4680、LGES 美国 LFP 项目的量产进展。
4. **追踪政策风险**：美国关税、中国出口管制、关键矿物许可、LFP 和石墨相关限制。
5. **用多源交叉验证**：只有 Tesla 文件、供应商公告、监管披露、海关/进出口数据、可信媒体互相印证后，才把某供应关系列为“确认”。
6. **建立“确认/待核实/传闻”三层清单**：避免把行业传闻误写成确定供应链关系。
