# v0.2 Alpha Plan — 从可运行切片到可复现研究工具

本文定义 v0.2-alpha 的产品和工程范围。它不是 Phase 2 的全部完成定义，而是一个可以发布、可以被外部研究员复现的中间版本。

## v0.2 定位

v0.2 的目标：

```text
让一个外部研究员在本地复现 NVIDIA 供应链研究，并看到：
1. 高等级事实边
2. evidence trace
3. changes / source health
4. component card
5. chain-first research workbench
```

v0.2 不追求：

```text
完整 25 节点覆盖
漂亮投资 dashboard
LLM 自动抽取
宏观/物流/矿产源大规模接入
```

## P0：不做不能 release

### 1. Apple Supplier List facility edges

当前代码状态：

```text
SupplierListReviewPayload 已经包含 facility_relation_hint = MANUFACTURES_AT
review apply 已经会从同一条 approved supplier-list row 写入 buyer BUYS_FROM supplier
review apply 同时会稳定创建 facility entity，并写入 supplier MANUFACTURES_AT facility
```

因此 v0.2 的剩余实现重点不是“再解析 PDF”，而是把这条 apply 路径批量跑起来，补齐 seed/alias，让足够多的 Apple supplier row 能通过 review/apply 变成真实 facility edges。

验收：

```text
[ ] 至少 50 条 `MANUFACTURES_AT` 或 facility-related Level 4 edge
[ ] 每条 edge 有 cite_text、locator、reviewer
[ ] 未 review 候选不入 graph
[x] review apply / batch apply 都落 ChangeRecord
[x] facility entity 有稳定 ID、country_or_region、source row provenance
```

### 2. SEC 10-Q / 8-K 接入

验收：

```text
[x] `sec-edgar` 能 plan/fetch/normalize 最新 10-Q
[x] `sec-edgar` 能 plan/fetch/normalize 最近若干 8-K
[ ] source monitor 记录 DOCUMENT_NEW / DOCUMENT_CHANGED / DOCUMENT_UNCHANGED
[ ] 至少 1 条新 evidence 或 change 能从 10-Q/8-K 触发
```

实现边界：`sec-edgar` adapter 的 `formTypes` 支持 `10-K / 10-Q / 20-F / 8-K`，`limit` 可让 plan 一次吐出多份最近 8-K task；pipeline shortcut 仍默认只跑 NVIDIA 10-K，10-Q / 8-K 的变化监控要等 source monitor 调度层接入。

### 3. `cli changes`

验收：

```text
[ ] `supplystrata changes --since <date>` 可读输出
[ ] `apply` / `review apply` / `deprecate` / Apple supplier import 都写 ChangeRecord
[ ] changes 输出能指向 edge_id / evidence_id / source_item_id
```

### 4. ComponentCard + `cli component`

验收：

```text
[ ] `supplystrata component memory --format markdown`
[ ] 输出 known_suppliers / known_consumers / unknown_map
[ ] 每条事实性陈述挂 evidence_id
[ ] 默认隐藏 Level 1-3 inferred 内容
```

### 5. EntityResolver golden set

验收：

```text
[ ] golden set >= 200
[ ] CI 跑通
[ ] 高歧义实体覆盖 Samsung / Foxconn / TSMC / 3M / Apple / Micron / SK
[ ] false merge 零容忍；宁可 unknown
```
